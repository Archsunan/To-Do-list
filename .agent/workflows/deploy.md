---
description: How to deploy the Streamline To-Do List application.
---

### Option 1: GitHub Pages (Recommended)
1. Initialize a git repository in the project folder.
2. Create a new repository on GitHub.
3. Push your code to GitHub.
4. Go to **Settings** > **Pages** and select the main branch as the source.

### Option 2: Vercel / Netlify
1. Drag and drop the `To do list` folder into the [Vercel](https://vercel.com) or [Netlify](https://netlify.com) dashboard.
2. The platform will automatically detect the static files and provide a deployment URL.

### Option 3: Local Server
If you want to run it locally with a server:
1. Run `npx serve .` in the terminal to start a local static server.
