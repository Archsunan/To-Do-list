# Streamline | Modern To-Do List

A premium, high-performance To-Do List application built with a modern Neobrutalist design aesthetic.

## Features

- **Multi-Theme Support**: Choose between Mono (Black & White), Indigo (Dark Mode), and Emerald (Light/Fresh) themes.
- **Glassmorphism & Neobrutalism**: A unique blend of modern design trends for a premium feel.
- **Persistent Storage**: Uses `localStorage` to save your tasks locally.
- **User Accounts**: Basic local authentication (Login/Signup) to keep your tasks separate.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Smooth Animations**: Interactive task actions with polished transitions.

## Technology Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom variables, Neobrutalist layout, and responsive styles.
- **JavaScript**: Pure Vanilla JS for logic and state management.
- **Bootstrap 5**: Used for layout utilities and basic components.

## Local Development

Simply open `home.html` in any modern web browser to run the application.
